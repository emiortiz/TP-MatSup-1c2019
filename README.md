# Documentation 
## Enviroment

Base system: Linux Fedora 29
Requirements:
 - python 3.7 or greater
 - pip 1.8 or greater

## Enviroment setup
### Install virtualenv

~~~
$ pip install --user virtualenv
~~~

### Clone repo

~~~
$ git clone <repo>
~~~

### Create a virtualenv and activate it

~~~
$ virtualenv venv
$ source venv/bin/activate
~~~

### Install requirements

~~~
$ pip install -r requeriments.txt
~~~
